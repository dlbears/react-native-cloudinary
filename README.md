# react-native-cloudinary

cloudinary sdks for react native

## Installation

```sh
npm install react-native-cloudinary
```

## Usage

```js
import Cloudinary from "react-native-cloudinary";

// ...

const result = await Cloudinary.multiply(3, 7);
```
